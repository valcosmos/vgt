## Install

```sh

# use yarn
yarn global add vgt

# use pnpm
pnpm i -g vgt

# use npm
npx i -g vgt

```

## Usage

```sh

# run this command, and u can get a vue3 template

vgt init demo -t project -tp vue3-template -f

# or just run this, u can choose the configuration u want

vgt init

```
